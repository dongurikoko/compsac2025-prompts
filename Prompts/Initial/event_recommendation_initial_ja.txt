# イベント推薦用初期プロンプトの内容

ユーザの投稿は以下です。
*** ユーザの投稿内容(例) ***
「昔の文献読むの楽しい」
「今からカラオケ行きたい人いる?」

最初に、ユーザの投稿から、ユーザの性格特性を推測します。
推測は、big five性格特性を基に行います。
ユーザの投稿から、協調性、誠実性、外向性、神経症傾向、開放性の各特性に対して、スコアづけを行います。
1 = 強く不同意
2 = 少し不同意
3 = 中立
4 = 少し同意
5 = 強く同意

逆スコアリング項目:
逆スコアリングが必要な項目（S1,S3,S6,S8）については、平均点を算出する際に、スコアを逆にした上で計算を行います。
S1: 著者は時々他人の欠点に気づく。
S2: 著者は利己的でなく他人を助ける。
S3: 著者は時々他人と口論する。
S4: 著者は他人を容易に許す。
S5: 著者は人を容易に信じる。
S6: 著者は時々他人に冷淡であることがある。
S7: 著者は皆に親切である。
S8: 著者は時々無礼であることがある。
S9: 著者は他人と協力するのが好きである。

S1~S9の各項目に対し、スコアとスコアをつけた理由を必ず出力してください。
詳しく、どの投稿を参考にしたか載せてください。
記載が見られない場合は中立 (スコア3)としてください。

上記のスコアに基づいて、著者の協調性のスコアを算出してください。
ここで、スコアの算出は、必ずPythonでプログラムの実行を行った上でスコアを出してください。
スコアの算出の計算式と、結果も書いてください。(例：○ + ○ = △ )

********** プログラム内容 **********
# 各項目のスコアをリストに入れる
協調性 = [, , ...]

# 逆スコアリングが必要な項目のインデックス
協調性_逆 = [0, 2, 5, 7]

# 逆スコアリングを適用する関数
def reverse_score(scores, reverse_indices):
    return [6 - scores[i] if i in reverse_indices else scores[i] for i in range(len(scores))]

# 各特性のスコアを逆スコアリングを適用した上で計算
協調性_スコア = sum(reverse_score(協調性, 協調性_逆)) / len(協調性)

# 各特性のスコアを表示
print(f"協調性: {協調性_スコア}")
********** プログラム内容終わり **********

次に、与えられたテキストからユーザの趣味趣向として適切なものを推測し、出力してください。

今度は、ユーザの投稿をもとに感情を推測してください。
ユーザーの投稿内容に対して、以下の1~4のステップでユーザの感情を理解します。
1. ユーザーが示している感情はなんですか？
2. その感情はどのような状況や考えから生じていますか？
3. ユーザーが感じている感情に共感し、深い理解を示してください。
4. ユーザーが抱える問題に対して、具体的なアドバイスや解決策を考えてください。
以上の1~4を元に、感情を推測し、その過程と結果を出力してください。

以上の推測した性格特性、趣味趣向、感情に基づいてさらにイベントの絞り込みを行い、ユーザにイベントを推薦します。
絞り込み方法は以下です。まず、性格特性をもとに分類し、その上で趣味趣向と感情を参考にイベントを推薦します。
○ 開放性
高い人 → 好奇心を刺激されるような新たな物事や、意外性のある物事へと取り組む活動を好む傾向にあるため、 ユーザの主な興味に少し遠いが関連性のある趣味に基づいたイベント推薦を行う。
低い人 → 安定思考で保守的な性質のため、人気、有名なイベントを推薦する。新しい物事は推薦しない。
○ 協調性
高い人 → 世間一般や周りに合わせる傾向があるため、人気、有名なイベントや、複数人で協力/一緒に取り組むことができる活動を推薦する。
低い人 → 上記は避ける。
○ 外交性
高い人 → アクティビティ寄りのイベントを推薦する。
低い人 → 個人の趣味趣向を重視して、イベントをさらに絞り込む。
○ 神経症傾向
高い人 → 不安定な性質があるため、バラエティに富んだイベントを推薦する。
○ 勤勉性
高い人→ 気持ちのリフレッシュによるモチベーションアップや達成感につながるイベントを好む傾向があるため、感情に基づいてさらにイベントの絞り込みを行う。
低い人→ 見どころがはっきりしたイベントを推薦する。また、飽きっぽく移り気な性格である傾向もあるため、様々なジャンルのイベントを推薦する。

以上のような方法で絞り込みを行い、適切だと考えられるイベントを３つ推薦してください。
推薦する際は、性格特性、感情、趣味趣向においてどのような理由でそのイベントを推薦したのか理由もあわせて教えてください。
